## 丕賱鬲胤亘賷賯

- **丕賱賲鬲噩乇** (賵丕噩賴丕鬲 丕賱毓賲賱丕亍): React + Vite (丕賱賲鬲噩乇 + 賱賵丨丞 丕賱鬲丨賰賲)
- **丕賱禺丕丿賲**: Express 5 + Node.js 24
- **賯丕毓丿丞 丕賱亘賷丕賳丕鬲**: PostgreSQL + Drizzle ORM
- **丕賱賮鬲乇丕鬲**: PNPM Monorepo

## 丕賱賲鬲睾賷乇丕鬲

### 丕賱賵丕噩賴丕鬲 (Vercel)

- **丕賱賲鬲噩乇**: `artifacts/techzone`
  - Build Command: `pnpm install && pnpm run build`
  - Output Directory: `dist`
  - Framework: Vite
  - Env: `VITE_API_URL=https://api.nexus-store.com`
- **賱賵丨丞 丕賱鬲丨賰賲**: `artifacts/techzone-admin`
  - Build Command: `pnpm install && pnpm run build`
  - Output Directory: `dist`
  - Framework: Vite
  - Env: `VITE_API_URL=https://api.nexus-store.com`

### 丕賱禺丕丿賲 (Railway / Render)

- **Root Directory**: `artifacts/api-server`
- **Start Command**: `pnpm install && pnpm run build && node dist/index.js`
- **Port**: `5000` (Railway/Render 賷禺鬲丕乇丕賳 丕賱亘賵乇鬲 鬲賱賯丕卅賷賸丕)
- **Env**:
  - `DATABASE_URL` (PostgreSQL connection string)
  - `SESSION_SECRET` (賳氐 毓卮賵丕卅賷 胤賵賷賱)
  - `NODE_ENV=production`
  - `COOKIE_SECRET` (賱賭賱賱賱賱賱賱)
  - `UPLOAD_DIR` (optional, for local storage)
