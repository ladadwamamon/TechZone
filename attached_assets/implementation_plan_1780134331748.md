# متجر TechZone للإلكترونيات والجيمنج - الخطة الشاملة النهائية

## نظرة عامة
بناء متجر إلكتروني متكامل ومتطور للإلكترونيات وقطع الكمبيوتر باستخدام **Next.js 14** مع تصميم Dark Gaming عصري يدعم اللغة العربية ويحقق مبيعات فعلية.

---

## التقنيات المستخدمة

| التقنية | الغرض |
|---------|--------|
| **Next.js 14** (App Router) | إطار العمل + SSR لدعم SEO الكامل |
| **Tailwind CSS** | التصميم والتنسيق |
| **Zustand** | إدارة الحالة (السلة، المستخدم، PC Builder) |
| **Framer Motion** | الحركات والانيميشن |
| **Lucide React** | الأيقونات |
| **React Hook Form + Zod** | نماذج الدفع والتحقق |
| **Fuse.js** | البحث الذكي مع تصحيح الأخطاء الإملائية |
| **next-seo** | Meta Tags + JSON-LD Schema |
| **nextjs-toploader** | مؤشر التحميل النيون بين الصفحات |
| **gray-matter + remark** | قراءة وتحويل مقالات MDX للمدونة |

## 0. 🌐 عناصر الواجهة الشاملة (Global UI Elements) ⭐ جديد

### 📢 شريط الإعلانات العلوي (Announcement Bar)
- شريط **رفيع جداً** فوق الـ Navbar متحرك (Marquee) أو ثابت
- أمثلة محتواه: *"شحن مجاني للطلبات فوق 500 شيكل"* أو *"خصم 10% بكود COBRA"
- زر إغلاق (X) ويحتفظ بإغلاقه خلال الجلسة (localStorage)
- قابل للتعديل من ملف الإعدادات
- **لون Neon** متوافق مع Primary Color الموقع

### ⏳ مؤشر التحميل بين الصفحات (Page Transition Progress Bar)
- خط رفيع **Neon** (لون `#00D4FF`) يظهر **أعلى الشاشة** عند التنقل بين أي صفحتين
- مكتبة: **`nextjs-toploader`** (خفيفة جداً)
- يعطي إحساس السرعة ويمنع شعور التعليق
- يُضاف **مرة واحدة** في `layout.tsx` الرئيسي

### ↑ زر العودة للأعلى (Scroll to Top Button)
- زر **عائم (Floating)** أسفل الشاشة يظهر فقط بعد التمرير 300px للأسفل
- **انيميشن سلسة** للظهور والاختفاء
- **Scroll Smooth** فوري لأعلى الصفحة
- تصميم Gaming مع Neon Glow
- يظهر فقط في صفحات الكتالوج والمدونة (ليس في Checkout)

### 🍪 إشعار ملفات الارتباط (Cookie Consent Banner)
- مستطيل **صغير** أسفل الشاشة بتصميم Glassmorphism
- يظهر **فقط** لأول زيارة (ثم يختفي للأبد بعد الموافقة)
- زران: **"قبول الجميع"** + **"إدارة التفضيلات"**
- رابط لسياسة الخصوصية
- ضروري لـ Google Analytics والـ Pixel
- يحفظ الاختيار في localStorage

## 1. 🗺️ صفحات التصفح والاكتشاف (Discovery & Browsing)

### 🏠 الصفحة الرئيسية (Home Page)
- **Hero Banner**: بانرات متحركة (Swiper) للعروض الكبرى مع Gradient أنيق
- **Flash Deals Section**: عروض محدودة مع عداد تنازلي (Countdown Timer)
- **Featured Categories**: شبكة الأقسام (شاشات، كروت شاشة، كونسول، إكسسوارات)
- **Best Sellers**: كاروسيل أفضل المنتجات مبيعاً
- **Bundles Section**: عرض باقات مميزة (PC Gaming Setup بسعر خاص)
- **Brands Showcase**: شريط العلامات التجارية المتحرك
- **Trust Badges**: ضمان، شحن سريع، إرجاع، دفع آمن
- **Newsletter**: اشتراك بالنشرة البريدية

### 🗂️ صفحة الأقسام (Categories Page)
- عرض **جميع الأقسام** بشبكة Grid أنيقة مع أيقونات وصور لكل قسم
- **أقسام رئيسية**: معالجات، كروت شاشة، شاشات، RAM، تخزين، كونسول، إكسسوارات، لابتوبات، شبكات
- **عدد المنتجات** في كل قسم
- **Hover Animation** جذاب على البطاقات

### 🔥 صفحة العروض (Deals & Flash Sales)
- **Hero خاص** بالعروض مع تصميم نار
- **Countdown Timer** لكل عرض محدود الوقت
- **فلترة العروض**: حسب نسبة الخصم، القسم، السعر
- **Progress Bar** يوضح كمية المتبقي ("تبقى 3 قطع فقط!")
- **Flash Deals** تتجدد يومياً (وهمياً)

## 2. 🛍️ صفحات الكتالوج والتسوق (Catalog & Shopping)

### 📋 صفحة قائمة المنتجات (PLP - Product Listing Page)
- تظهر عند الضغط على أي قسم
- **فلاتر متقدمة** (السعر Range Slider، الماركة، التوفر، الضمان، التقييم)
- **شريط الترتيب**: من الأرخص، الأغلى، الأعلى تقييماً، الأحدث
- **عرض Grid / List** قابل للتبديل
- **بطاقة المنتج** تحتوي على: صورة، اسم، سعر، تقييم نجوم، **بادج الضمان** 🛡️، زر "أضف للسلة" سريع
- **Skeleton Loading** أثناء التحميل
- **Pagination** أو Infinite Scroll

### 📱 صفحة تفاصيل المنتج (PDP - Product Detail Page)
- **معرض صور تفاعلي** مع Zoom + Thumbnail
- **السعر** + سعر قبل الخصم + نسبة التوفير
- **📌 رقم الصنف (SKU / Part Number)** ⭐ ظاهر بوضوح تحت اسم المنتج - الزبون المتخصص يبحث برقم القطعة ليتأكد من النسخة
- **🛡️ بادج الضمان** بارز جداً (مثال: "ضمان سنتان - وكيل رسمي")
- **المواصفات التقنية الدقيقة** (جدول منظم)
- **اختيار الكمية** + حالة التوفر
- **حالة نفاد الكمية (Out of Stock State)** ⭐:
  - زر "أضف للسلة" يتحول للون **الرمادي غير قابل للضغط**
  - حقل **"أعلمني عند التوفر"** (Notify me) لجمع الإيميل
  - بادج أحمر **"غير متوفر"** بجانب السعر
- **3 أزرار رئيسية**:
  - 🛒 أضف للسلة
  - ⚡ اشتري الآن
  - 💬 **استفسر عبر واتساب** (رابط جاهز باسم المنتج + السعر)
- **أزرار المشاركة (Social Share)** ⭐ أيقونات صغيرة: 📱 واتساب | 🐦 تويتر | 📋 نسخ الرابط
- **تبويبات**: الوصف | المواصفات | المراجعات | سياسة الشحن
- **التقييمات والمراجعات** مع توزيع النجوم
- **منتجات ذات صلة** + "اشتراه معه أيضاً"
- **JSON-LD Schema** ⭐ يُضاف برمجياً لكل منتج (نجوم + سعر في Google)

### 🔍 صفحة نتائج البحث (Search Results Page)
- بحث فوري مع **Autocomplete** و **Trending Searches**
- **تصحيح الأخطاء الإملائية** (Fuzzy Search بـ Fuse.js): "سني" → يعرض "Sony"
- **"هل تقصد: ..."** عند كتابة خطأ
- فلترة النتائج + ترتيبها
- **عدد النتائج** وإظهار كلمة البحث

## 2.5 💨 حالات الواجهة الفارغة (Empty States) ⭐ جديد

> [!TIP]
> الشاشات الفارغة هي الفرصة الدهبية لتحويل الإحباط إلى فعل - لا تتركها بيضاء أبدا.

### 🛒 سلة فارغة (Empty Cart)
- أيقونة **سلة حزينة** بتصميم لطيف Gaming
- رسالة: *"سلتك فارغة... لنبدأ المهمة!"*
- **زر كبير**: "ابدأ التسوق الآن"
- عرض **3 منتجات موصى بها** أسفل الرسالة

### 🔍 بحث بدون نتائج (Zero Search Results)
- أيقونة معبّرة (مثلاً جهاز بخير تالف)
- رسالة أفضل من "لا يوجد": *"لم نجد ما تبحث عنه، لكن تفقد هذه المنتجات المميزة"*
- اقتراح **4 منتجات** ذات صلة أو Best Sellers
- زر: "تصفح جميع المنتجات"

### ❤️ مفضلة فارغة (Empty Wishlist)
- أيقونة قلب Gaming
- رسالة: *"قائمة أمنياتك فارغة - أضف ما يعجبك واحتفظ به!"*
- **زر**: "اكتشف المنتجات"

### 📦 طلبات فارغة (Empty Order History)
- رسالة: *"لم تطلب شيئاً بعد - حان الوقت!"*
- **زر كبير**: "ابدأ التسوق"

## 3. 💳 مسار الشراء (Checkout Flow)

### 🛒 السلة الجانبية (Drawer / Slide-over Cart)
- تنزلق بخفة من جانب الشاشة عند إضافة أي منتج
- **لا تغادر الصفحة الحالية**
- تعديل الكميات وحذف المنتجات
- **الإجمالي الفوري**
- زرا: "متابعة التسوق" + "الذهاب للسلة"

### 🗂️ صفحة السلة الرئيسية (Shopping Cart Page)
- مراجعة كاملة لجميع المنتجات
- تعديل الكميات + حذف + إضافة للمفضلة
- **حقل كود الخصم (Promo Code)** مع تأكيد فوري
- **تقدير تكلفة الشحن** (اختيار المدينة)
- **ملخص الطلب**: المجموع، الخصم، الشحن، **الإجمالي النهائي**
- اقتراح "Bundle": "أكمل الباقة بـ X ريال فقط وتوفر أكثر"
- زران: "متابعة التسوق" + "إتمام الطلب"

### 💳 صفحة إتمام الطلب (One-Page Checkout)
- **صفحة موحدة** خالية من أي تشتيت (بدون Header/Footer كاملين)
- **خيار عند البداية**:
  - 👤 تسجيل الدخول (للأعضاء)
  - 🚀 **الشراء كزائر (Guest Checkout)** بدون إلزام بحساب
- **حقول مطلوبة**: الاسم الكامل، رقم الجوال، المدينة، تفاصيل العنوان
- **طرق الدفع**: الدفع عند الاستلام، بطاقة (Mock)، تحويل بنكي
- **ملخص الطلب** في الجانب (Sticky)
- **زر التأكيد** مع Loading State

### ✅ صفحة تأكيد الطلب (Order Success / Thank You Page)
- **رقم الطلب (Order ID)** بارز وقابل للنسخ
- **رسالة شكر** بتصميم احتفالي (Confetti Animation)
- ملخص ما تم طلبه
- **زر تتبع الطلب**
- **زر الرجوع للمتجر**
- للزوار: "**إنشاء حساب الآن** لمتابعة طلباتك بسهولة" (اختياري)

## 4. 👤 مساحة المستخدم (User Account & Dashboard)

### 🔐 صفحات المصادقة (Auth Pages)
- **تسجيل الدخول (Login)**: إيميل + كلمة مرور
- **إنشاء حساب (Register)**: الاسم، الإيميل، كلمة المرور، رقم الجوال
- **استعادة كلمة المرور (Forgot Password)**: إرسال كود للإيميل
- تصميم موحد بخلفية Glassmorphism

### 📊 لوحة تحكم الحساب (My Account Dashboard)
- **ملخص سريع**: عدد الطلبات، المفضلة، العناوين المحفوظة
- **آخر طلب** وحالته
- **روابط سريعة** لجميع أقسام الحساب

### 📦 الطلبات السابقة (Order History)
- قائمة بكل الطلبات مع: رقم الطلب، التاريخ، الإجمالي، **الحالة** (قيد المعالجة / تم الشحن / مكتمل / ملغي)
- **تفاصيل كل طلب** عند الضغط عليه
- **إعادة الطلب** بنقرة واحدة

### 🔎 تتبع الطلب (Order Tracking)
- **صفحة عامة** بدون تسجيل دخول (مهمة للزوار!)
- إدخال **رقم الطلب + رقم الجوال**
- **Timeline مرئي**: قيد المعالجة ← تم التأكيد ← تم الشحن ← جاري التوصيل ← تم التسليم
- رابط تتبع شركة الشحن

### ❤️ قائمة المفضلة (Wishlist)
- حفظ المنتجات للشراء لاحقاً
- إضافة مباشرة للسلة من المفضلة
- **مشاركة قائمة المفضلة** برابط

### 🏠 إدارة العناوين (Saved Addresses)
- إضافة / تعديل / حذف عناوين التوصيل
- **تعيين عنوان افتراضي**
- يُستخدم تلقائياً عند الدفع لتسريع الطلبات القادمة

### ⚙️ إعدادات الحساب (Profile Settings) ⭐ جديد
- **تغيير الاسم** الكامل
- **تغيير البريد الإلكتروني** (مع تأكيد بكلمة المرور)
- **تغيير كلمة المرور** (القديمة + الجديدة + التأكيد)
- **تحديث رقم الجوال**
- **حذف الحساب** (مع تحذير تأكيد)
- تصميم موحد مع باقي صفحات الحساب

## 8. 🧭 مكوّن Breadcrumbs (تنقل الصفحات) ⭐ جديد

مكوّن **أساسي** يظهر في **كل الصفحات الداخلية** بدون استثناء:

```
الرئيسية > الأقسام > شاشات > شاشة ASUS TUF 144Hz
الرئيسية > الماركات > Razer > Razer DeathAdder
الرئيسية > المدونة > كيف تختار كرت الشاشة؟
```

- **تصميم مدمج** في الـ Layout لا يحتاج إضافة يدوية في كل صفحة
- **JSON-LD BreadcrumbList** مرفق تلقائياً لدعم SEO
- **قابل للضغط** على كل مستوى للتنقل السريع
- **مخفي** في الصفحة الرئيسية والـ Auth pages والـ Checkout

## 9. 🗺️ Sitemap & Robots.txt ⭐ جديد

### sitemap.xml (توليد برمجي في Next.js)
```typescript
// app/sitemap.ts
// يُولّد تلقائياً روابط:
- الصفحات الثابتة (/, /categories, /deals, /brands, /blog...)
- كل صفحة منتج (/products/[id]) - ديناميكي من Mock Data
- كل صفحة ماركة (/brands/[slug])
- كل مقال (/blog/[slug])
// يُحدَّث عند كل build
```

### robots.txt (توليد برمجي)
```typescript
// app/robots.ts
User-agent: *
Allow: /
Disallow: /account/
Disallow: /checkout/
Disallow: /auth/
Sitemap: https://techzone.com/sitemap.xml
```

## 14. 🌐 دعم العربية بذكاء (RTL Excellence) ⭐

### 💬 الجمع والمثنى الديناميكي (Arabic Smart Pluralization)
دالة برمجية تلقائية:
```typescript
function pluralize(n: number, word: string): string {
  if (n === 1) return `${n} ${word}`          // 1 منتج
  if (n === 2) return `${word}ان`             // منتجان
  if (n >= 3 && n <= 10) return `${n} ${word}ات` // 5 منتجات
  return `${n} ${word}اً`                    // 25 منتجاً
}
```
الاستخدام: السلة، نتائج البحث، عدد المنتجات، عدد التقييمات

### 🔄 انعكاس الأيقونات الاتجاهية (Icon Mirroring for RTL)
- سهم "التالي" ينعكس ليصبح "السابق" في RTL
- سهم "chevron" وأيقونات Slider تنعكس تلقائياً
- صف CSS: `[dir="rtl"] .icon-directional { transform: scaleX(-1); }`
- أيقونات لا تتأثر: قلب، عربة، نجمة (لا اتجاهية)

## 19. 🕵️‍♂️ الحماية والأمان (Security & Anti-Abuse) ⭐

### 🛡️ منع الطلبات الوهمية (Rate Limiting & Honeypots)
- **Rate Limiting**: بعد 3 طلبات من نفس الجهاز في دقيقة → رسالة "يرجى الانتظار"
- **Honeypot Field**: حقل مخفي CSS في نموذج الدفع
  - البشر لا يرونه = لا يعبئونه
  - البوت يعبئه تلقائياً = رفض الطلب صمتاً
- يحمي من هجمات السبام والطلبات الوهمية

### 📧 إخفاء الإيميلات (Email Obfuscation)
- **لا تكتب الإيميل كنص عادي** في HTML (info@store.com)
- تشفير برمجي: يحتفظ بالإيميل مشفراً ويفكه JavaScript للمتصفح
- أو استخدام **CSS عكسي** + محتوى خاص بالـ `::before`
- يحمي من روبوتات السبام
